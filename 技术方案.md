# 基于 C++ 与 OpenGL 的本地太阳系实时渲染 — 技术方案

## 1. 项目概述

### 1.1 项目目标

构建一个**不依赖商业游戏引擎**的本地太阳系实时渲染程序，使用 C++17 与 OpenGL 4.6 Core Profile 完成渲染管线，实现太阳、八大行星的三维可视化、公转自转模拟、自由摄像机飞行与基础光照系统。

### 1.2 核心原则

| 原则 | 说明 |
|------|------|
| **无商业引擎** | 不引入 Unity / Unreal 等商业引擎，全部手写渲染管线 |
| **可移植性** | 所有第三方图形库（GLFW、GLAD、GLM、stb_image、ImGui）以源码形式保存在项目 `thirdparty/` 目录下 |
| **可扩展性** | 模块化架构，预留大气散射、LOD、多线程资源管理等扩展点 |
| **学习导向** | 代码结构清晰，适合作为现代图形学学习参考 |

## 4. 渲染管线设计

### 4.1 整体流程

```
ImGui NewFrame → ESC/输入处理 → 场景更新 → Forward Render → ImGui Render → Swap
```

### 4.2 Forward Rendering

单 Pass 绘制，顺序：轨道线 → 太阳 → 八颗行星 → ImGui UI 叠加

### 4.3 着色器设计

**行星着色器** (planet.vert/frag):
- Blinn-Phong 光照（Ambient + Diffuse + Specular）
- 单点光源（太阳位于原点），距离衰减
- 无纹理时使用 FragPos 驱动动态着色（运动可视化验证）

**太阳着色器** (sun.vert/frag):
- 自发光 + 菲涅尔边缘渐变
- `uLightColor` uniform 控制色调

**轨道着色器** (orbit.vert/frag):
- 纯色 + 半透明，`GL_LINES` 绘制

**天空盒着色器** (skybox.vert/frag):
- Cubemap 采样，`GL_LEQUAL` 深度测试，`pos.xyww` 保证远平面

## 5. 天体系统设计

### 5.1 数据结构（实际实现）

```cpp
struct CelestialParams {
    std::string name;
    float       radius;               // 渲染半径（世界单位）
    float       orbitRadius;          // 轨道半径（0 = 恒星）
    float       orbitPeriodDays;      // 真实公转周期（地球日）
    float       rotationPeriodHours;  // 真实自转周期（小时，负 = 逆行）
    float       axialTilt;            // 轴倾角（度）
    std::string texturePath;
};
```

### 5.2 真实天文数据

| 行星 | 公转周期(地球日) | 自转周期(小时) | 轴倾角(°) |
|------|-----------------|---------------|-----------|
| 水星 | 87.97 | 1407.5 | 0.03 |
| 金星 | 224.7 | -5832.5 (逆行) | 2.64 |
| 地球 | 365.25 | 23.93 | 23.44 |
| 火星 | 686.98 | 24.62 | 25.19 |
| 木星 | 4332.59 | 9.93 | 3.13 |
| 土星 | 10759.22 | 10.66 | 26.73 |
| 天王星 | 30688.5 | -17.24 (逆行) | 97.77 |
| 海王星 | 60182.0 | 16.11 | 28.32 |

### 5.3 速度公式

- **公转角速度**: `ω_orbit = 2π × DAYS_PER_SECOND ÷ periodDays` (rad/s sim-time)
- **自转角速度**: `ω_rotation = 2π × DAYS_PER_SECOND × 24 ÷ periodHours` (rad/s sim-time)
- 默认 `DAYS_PER_SECOND = 30`（1 真实秒 = 30 模拟地球日），地球约 12 秒公转一圈

### 5.4 模型矩阵（关键修正）

```cpp
// 正确顺序（GLM 右乘：model = model * newMatrix）
// 顶点变换 v' = R_orbit * T_orbit * R_tilt * R_self * v
model = glm::rotate(model, m_orbitAngle, Y);           // 1. 公转
model = glm::translate(model, (orbitRadius, 0, 0));    // 2. 轨道平移
model = glm::rotate(model, axialTilt, Z);              // 3. 轴倾角
model = glm::rotate(model, m_rotationAngle, Y);        // 4. 自转
```

> 早期实现错误地使用了 `T * R` 顺序，导致行星被钉在固定位置只自转不公转。修正为 `R * T` 后坐标正确。

### 5.5 缩放策略

- 轨道距离与天体尺寸使用独立的艺术缩放值（`Constants.h`）
- 太阳半径 80 单位，地球轨道 180 单位，地球半径 3 单位
- 后续可扩展对数缩放

## 6. 摄像机系统

### 6.1 自由飞行模式

- 欧拉角 + 方向向量法，无万向锁（pitch 限制 -89° ~ 89°）
- 移动速度与 `rawDeltaTime` 挂钩，帧率无关
- 面板打开时 `if (!showSettings)` 屏蔽所有摄像机输入

### 6.2 窗口适配

- `glfwSetFramebufferSizeCallback` 注册回调，实时更新 `glViewport`
- 每帧 `updateFramebufferSize()` 同步窗口尺寸到 `m_width/m_height`
- `getAspectRatio()` 返回真实宽高比，投影矩阵自动适配

## 7. 光照系统

### 7.1 Phase 1: Blinn-Phong

- 唯一光源：太阳位于世界原点
- Ambient: 环境光，可通过 ImGui 面板调节 (0 ~ 0.5)
- Diffuse: `max(dot(N, L), 0)`
- Specular: `pow(max(dot(N, H), 0), 32) × 0.5`
- 线性 + 二次距离衰减

### 7.2 Phase 2 预留

- 阴影映射、HDR/Bloom、大气散射

## 8. 资源管理

### 8.1 Mesh

- `SphereMesh::generate(radius, sectors, stacks)` 程序化生成 UV 球体
- 每个天体独立持有自己的 Mesh 实例
- 轨道线使用 256 段 `GL_LINES`

### 8.2 纹理

- `Texture` 封装 stb_image 加载，支持 JPEG/PNG/sRGB
- `CelestialBody` 构造函数检查纹理路径，缺失时自动回退程序化着色

### 8.3 Shader

- `Shader` 类封装编译链：`glCreateShader → glCompileShader → glCreateProgram → glLinkProgram`
- 编译/链接错误通过 `glGetShaderInfoLog` / `glGetProgramInfoLog` 获取完整日志

## 9. 主循环与时间系统

### 9.1 主循环

```
while (!window.shouldClose() && !wantsQuit) {
    time.tick();
    input.update();
    ImGui 帧开始;
    ESC 切换面板（边缘触发）;
    顶栏渲染（FPS、dt、时间倍速、地球公转角）;
    控制面板渲染;
    面板开关转移检测（保存/恢复时间 + 光标模式）;
    键盘/鼠标输入处理（面板关闭时）;
    场景更新;
    3D 渲染;
    ImGui 帧结束;
    swapBuffers + pollEvents + updateFramebufferSize;
}
```

### 9.2 时间系统

```cpp
class Time {
    float m_deltaTime;      // 原始帧间隔 (s)，来自 glfwGetTime()
    float m_timeScale;      // 时间倍速 0 ~ 10（0 = 暂停）
    float m_elapsedTime;    // 模拟时间累计
};
// getDeltaTime() = m_deltaTime * m_timeScale
// getRawDeltaTime() = m_deltaTime
```

### 9.3 控制面板行为

- **ESC 边缘触发**：按下瞬间切换，不重复触发
- **打开时**：保存当前时间倍速 → 暂停 → 显示鼠标 → 锁定摄像机
- **关闭时**：恢复时间倍速 → 隐藏鼠标 → 解锁摄像机
- **转移检测在面板渲染之后**：覆盖 ESC / Hide 按钮 / X 按钮三种关闭路径

## 10. ImGui 集成

### 10.1 初始化顺序

```
Window → GLAD → Input → ImGui::CreateContext → ImGui_ImplGlfw_Init → ImGui_ImplOpenGL3_Init
```

Input 在 ImGui 之前初始化，避免 `glfwSetWindowUserPointer` 和 GLFW 回调冲突。

### 10.2 输入隔离

- `io.WantCaptureKeyboard` → 键盘控制屏蔽
- `io.WantCaptureMouse` → 鼠标旋转/滚轮屏蔽
- `!showSettings` → 面板打开时额外屏蔽所有输入

## 11. 构建与运行

### 11.1 构建

```bash
mingw32-make all     # 编译 GLFW + GLAD + ImGui + 项目源码 → build/SolarSystem.exe
mingw32-make run     # 编译并运行
mingw32-make clean   # 清理构建产物
```

### 11.2 运行

将 `build/SolarSystem.exe` 与 `build/assets/` 放在同一目录，双击运行。无外部依赖，所有库静态链接。

### 11.3 环境要求

| 组件 | 要求 |
|------|------|
| OS | Windows 10/11 |
| GPU | 支持 OpenGL 4.6（Intel HD 5000+ / NVIDIA Kepler+ / AMD GCN+） |
| 运行时 | 无需安装 VC++ Redist 或 .NET |
